# LiveStreamFetcher

多平台直播视频流获取工具，输入直播间链接，一键提取可用直播流。

## 功能特性

- **多平台支持**：抖音、快手、小红书、淘宝直播、YY直播、微信视频号
- **多格式输出**：自动提取 M3U8 / FLV / MP4 直播流链接
- **一键复制**：点击即复制流地址，方便粘贴到 OBS / VLC / ffplay
- **OBS 本地代理**：淘宝直播等需要鉴权的流，自动启动本地代理转发，OBS 直接填本地地址
- **登录状态管理**：快手、淘宝、小红书支持扫码登录，状态栏实时显示登录状态
- **嵌入式浏览器**：内置 Chromium，无需依赖系统浏览器，拷贝即用
- **启动密码验证**：云端密码，实时生效

## 支持平台

| 平台 | 解析方式 | 登录要求 | 备注 |
|------|---------|---------|------|
| 抖音 | 请求 + JS | 不需要 | - |
| 快手 | Playwright | 扫码登录 | 浏览器自动化解析 |
| 小红书 | Playwright | 扫码登录 | SPA 动态加载，监听 API 响应 |
| 淘宝直播 | Playwright | 扫码登录 | 需本地代理注入 Referer |
| YY直播 | Playwright | 不需要 | - |
| 微信视频号 | 工具辅助 | 扫码获取 | - |

## 快速开始

### 下载

前往 [Releases](https://github.com/Dragon617/LiveStreamFetcher/releases) 下载最新版本 EXE。

### 使用

1. 双击运行 `LiveStreamFetcher.exe`
2. 输入启动密码
3. 粘贴直播间链接到输入框
4. 点击「获取流链接」
5. 点击流卡片上的复制按钮，粘贴到 OBS / VLC 即可

> 淘宝直播：点击橙色「OBS」按钮，使用代理地址。

### 登录

快手、小红书、淘宝直播首次使用需扫码登录：

1. 状态栏点击对应平台登录状态标签
2. 弹出浏览器窗口，扫码登录
3. 登录成功后自动跳回解析页面

## 技术栈

- Python 3.14
- tkinter GUI
- Playwright（浏览器自动化）
- yt-dlp（抖音等平台流提取）
- ffmpeg（HEVC 转码代理）
- PyInstaller（打包）
